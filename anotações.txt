Curso de programação do ProgBr - HTML

Como navegar entre abas html:

Usamos a TAG <a> para podermos navegar de uma pagina para outra. 
Ex:

<a href="sobre.html">Programador Br</a>


E com isso nós podemos navegar entre as páginas HTML
